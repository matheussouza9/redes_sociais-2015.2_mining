# Redes sociais 2015.2 - Engenharia de Software - UFC Quixadá

Ps.: Recomendada a instalação via pip, pois essas bibliotecas também possuem suas dependências.

	]
	
	Bibliotecas: [

		'pony 0.6.2'
	]
}
